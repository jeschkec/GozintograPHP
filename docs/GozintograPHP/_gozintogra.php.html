defaultErrorHandler [line 47]

int defaultErrorHandler( int $errNumber, string $errMessage, string $errFile, int $errLine)

The default error handler defaultErrorHandler writes the error message to STDERR as a csv string (separator is ;, enclosing is ").

The string contains:

  • An ISO-8601-formatted date, e.g. 2008-11-21T13:30:53+01:00
  • The error level
  • The error message
  • The file name, in which the error occured
  • The line number, on which the error occured
